<h3>Инструкция</h3>
<br />
<b>Шорткод</b>
<br />
<i>[comments]</i> - Шорткод самостоятельно определяет товар, на котором запускается, и выводит оставленные пользователями комментарии с формой для публикации новой записи.
<br />
<br />

<b>Пример использования шорткода в файле шаблона карточек товаров /product.php</b>
<br />
<p>Официальные шаблоны уже имеют поддержку данного плагина, и в карточках товаров уже встроен необходимый шорткод. Чтобы увидеть работу плагина в действии, перейдите в любой товар на публичной части сайта и откройте вкладку "Комментарии" под изображениями продукта. Инструкция, приведенная ниже, предусмотрена для того случая, если шаблон уникальный:</p>
<p>
Для подключения шорткода, вставьте нижеприведенный код в файл шаблона /views/product.php в самый конец файла.
<br />
<br />
&lt;div style=&quot;margin:20px;&quot;&gt;<br/>
&lt;p&gt;[comments]&lt;/p&gt;<br/>
&lt;/div&gt;
<br />
<br />
<i>Данный код предназначен для демонстрации работы плагина, для более эстетичного вида необходимо настраивать таблицы стилей для каждого шаблона индивидуально.</i>
</p>

<br />
<br />
<b>Настройки</b>
<br />
<p>
На странице управления плагином находится панель настроек, вызываемая кнопкой <i>"Настройки"</i>, которая содержит опцию включения загрузки изображений и поля для ее настройки.
</p>
<br />
<b>Публикация отзыва</b>
<br />
<ul>
<li>Каждый оставленный отзыв перед публикацией отправляется на премодерацию. Одобрить, отредактировать и удалить отзыв можно на странице настроек плагина в панели администратора.</li>
<li>Если в опциях будет подключена загрузка изображений, то соответствующая возможность появится на форме. </li>
<li>Если пользователь не авторизован, то будет необходимо еще указать свое имя и email.</li>
</ul>
<br />
<b>Премодерация</b>
<br />
<p>
Прежде чем отзыв от пользователя будет добавлен к товару, его будет необходимо проверить в панели администратора. Инструменты премодерации:
</p>
<ul>
    <li><i>Редактирование</i>: форма для редактирования и одобрения отзыва</li>
    <li><i>Удаление</i>: администратор или модератор могут удалить запись вместо того что бы ее одобрить</li>
</ul>
<br />
<b>Информер</b>
<br />
<p>
Вверху панели администратора есть информер плагина, который отображает количество требующих премодерацию отзывов. Клик по информеру вызовет страницу управления плагином, где можно будет увидеть список всех отзывов.
</p>
